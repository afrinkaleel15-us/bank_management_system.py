# 🏦 Bank Management System

A beginner-friendly Python project that simulates basic banking operations.

## Features

- Deposit Money
- Withdraw Money
- Check Balance
- Exit

## Technologies Used

- Python 3

## How to Run

```bash
python bank_management_system.py
```

## Menu

```
1. Deposit
2. Withdraw
3. Check Balance
4. Exit
```

## Example

```
Deposit: ₹5000

Current Balance: ₹5000

Withdraw: ₹1000

Current Balance: ₹4000
```
